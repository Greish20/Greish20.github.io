Examen Parcial 2

Documento correspondiente al Segundo Examen Parcial de la materia Herramientas de Programación IV.

Descripción de la actividad:

Este examen evaluó el diseño de soluciones móviles y la gestión de infraestructura de servidores:

  • Parte I (Caso de Estudio - Chinos Café): Desarrollo de una aplicación móvil (CRUD) para la gestión de pedidos de pizzas y combos, incluyendo sincronización y replicación de datos entre bases de datos y confirmaciones automáticas vía JSON/correo.
  • Parte II (Red LAN Inalámbrica): Diseño y configuración de una topología de red para la conexión entre dos centros universitarios, implementando servicios DHCP, enrutamiento y servidores web Linux en entornos simulados.
  • Parte III (Seguridad y Ataques): Implementación de medidas de seguridad y ataques controlados, incluyendo la configuración de un servidor web con NGINX (HTTPS), ejecución de ataques de fuerza bruta (Hydra) y escaneo de vulnerabilidades (OWASP ZAP) para fortalecer la infraestructura.
