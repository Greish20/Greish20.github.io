Ejercicio Práctico 3

Descripción de la actividad:

Este ejercicio práctico integró diversas competencias técnicas en tres etapas:

  • Parte I (Caso de Estudio - Chiriquí Eats): Desarrollo de un prototipo CRUD para gestión de pedidos, implementación de replicación de datos entre bases de datos y configuración de un servidor Apache en Linux para la distribución del APK.
  • Parte II (Red LAN Inalámbrica): Configuración física y lógica de una red inalámbrica con servicios DHCP, DNS y pruebas de conectividad mediante comandos de red.
  • Parte III (Seguridad y Ataques): Simulación de un escenario de ataque DDoS, mitigación de vulnerabilidades y verificación del tráfico de red utilizando herramientas de monitoreo como Wireshark.
