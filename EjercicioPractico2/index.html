Ejercicio Práctico 2

Descripción de la actividad:

Este ejercicio se dividió en dos partes fundamentales:

  • Desarrollo de App: Creación de una aplicación móvil para "Mobil Botica", incluyendo menú de compañías telefónicas, gestión de recargas y registro de transacciones.
  • Simulación de Red LAN: Configuración y conectividad de red entre sucursales utilizando Cisco Packet Tracer.
